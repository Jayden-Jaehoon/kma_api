# Geodata 파일 설명

이 디렉토리는 기상 데이터 처리에 필요한 지리 정보 데이터(Geodata)를 포함하고 있습니다. 행정동 경계 데이터와 기상청 격자 시스템 간의 매핑 데이터를 포함합니다.

## 확장자별 설명

| 확장자 | 설명 |
| :--- | :--- |
| **.shp** | **Shapefile (Geometry)**: 지리적 형상(점, 선, 면) 정보를 담고 있는 주 파일입니다. |
| **.shx** | **Shapefile Index**: `.shp` 파일의 형상 데이터를 빠르게 검색하기 위한 인덱스 파일입니다. |
| **.dbf** | **dBase Database**: 지리적 형상에 대응하는 속성 정보(예: 행정동 명칭, 코드 등)를 표 형식으로 저장합니다. |
| **.prj** | **Projection**: 데이터의 좌표계(Coordinate System) 및 투영법 정보를 텍스트 형식으로 저장합니다. |
| **.cpg** | **Code Page**: `.dbf` 파일의 텍스트 인코딩 정보(예: UTF-8, CP949)를 명시합니다. |
| **.parquet** | **Apache Parquet**: 컬럼 기반의 효율적인 데이터 저장 형식으로, 대용량 데이터를 압축하여 빠르게 읽고 쓸 수 있습니다. |
| **.nc** | **NetCDF (Network Common Data Form)**: 다차원 배열 데이터를 저장하기 위한 형식으로, 기상 및 해양학 분야에서 위경도 기반 격자 데이터를 저장할 때 주로 사용됩니다. |
| **.md** | **Markdown**: 텍스트 기반의 문서 파일로, 현재 보고 계시는 설명 파일입니다. |

## 주요 파일 설명

### 1. 격자-행정동 매핑 데이터 (`grid_to_hjd.parquet`)
- **설명**: 기상청의 격자 지점과 행정동 코드(HJD_CD) 간의 관계를 정의한 데이터입니다. 기상 데이터를 행정동 단위로 집계하거나 매핑할 때 사용됩니다.
- **기준 연도**: 2022년 4분기 행정동 Shapefile (BC카드 연구 기간 기준)

### 2. 격자-법정동 매핑 데이터 (`grid_to_emd_umd.parquet`)
- **설명**: 기상청의 격자 지점과 법정동 코드(EMD_CD) 간의 관계를 정의한 데이터입니다.
- **기준 Shapefile**: 국가공간정보포털 법정경계(읍면동) 17개 시도

### 3. 통합 매핑 데이터 (`grid_to_unified.parquet`)
- **설명**: 행정동(HJD_CD)과 법정동(EMD_CD)을 하나의 격자점에 동시에 매핑한 통합 테이블입니다. `--region-type both` 사용 시 참조됩니다.

### 4. 지상 격자 위경도 데이터 (`sfc_grid_latlon.nc`)
- **설명**: 지상(Surface) 격자 시스템의 각 지점에 대한 위도(Latitude)와 경도(Longitude) 정보를 담고 있는 데이터입니다.
- **다운로드**: [기상청 API허브](https://apihub.kma.go.kr/) 융합기상 탭 > 고해상도 격자자료 위경도

## 매핑 결과 참고치

격자(2049x2049 = 약 4.2M점)는 한반도 주변 전체(바다, 북한, 인접국 포함)를 커버합니다.
남한 육지에 해당하는 격자점만 행정구역에 매핑되므로, 대부분의 격자점은 매핑 실패(NaN)가 정상입니다.

| 항목 | 값 | 비고 |
|------|-----|------|
| 전체 격자점 | ~4,198,401 | 2049 x 2049 |
| HJD 매핑 성공 | ~386,056 (~9.2%) | 행정동 3,521개, 동당 평균 ~110 격자점 |
| BJD 매핑 성공 | ~384,318 (~9.2%) | 법정동 5,369개, 동당 평균 ~72 격자점 |
| 통합 양쪽 매핑 | ~383,842 | HJD/BJD 모두 성공한 격자점 |
| HJD만 매핑 | ~2,526 | 경계 미세 차이에 의한 edge effect |
| BJD만 매핑 | ~476 | 동일 |
| 매핑 실패 | ~3,812,345 (~90.8%) | 해양, 북한, 경계 밖 격자점 (집계 시 자동 제외) |

### 매핑 성공률 9%가 정상인 이유

격자 도메인의 전체 면적 대비 남한 육지 면적을 계산하면 매핑 성공률과 거의 일치합니다.

| 항목 | 값 |
|------|-----|
| 격자 전체 커버리지 | 2049 x 2049 x 500m = 약 1,024km x 1,024km = **~1,048,576 km²** |
| 남한 육지 면적 | **~100,210 km²** |
| 면적 비율 | 100,210 / 1,048,576 = **~9.6%** |
| 실제 매핑 성공률 | **~9.2%** |

0.4%p 차이는 해안선 경계의 격자점이 Shapefile 폴리곤 밖으로 빠지는 edge effect로 설명됩니다.

격자는 남한만 커버하는 것이 아니라 한반도 전체 + 주변 해역(동해, 서해, 남해, 북한, 일부 중국/일본 인접 영역)을 포함합니다. 따라서 대부분의 격자점은 바다이거나 남한 밖이며, 남한 육지에 해당하는 격자점만 행정동/법정동에 매핑됩니다.

> 위 수치는 2022년 4분기 행정동 + 2026년 2월 법정동 Shapefile 기준 참고치이며, Shapefile 버전에 따라 소폭 달라질 수 있습니다.
