# Geodata 파일 설명

이 디렉토리는 기상 데이터 처리에 필요한 지리 정보 데이터(Geodata)를 포함하고 있습니다. 행정구역 경계 데이터와 기상청 격자 시스템 간의 매핑 데이터를 포함합니다.

## 확장자별 설명

| 확장자 | 설명 |
| :--- | :--- |
| **.shp** | **Shapefile (Geometry)**: 지리적 형상(점, 선, 면) 정보를 담고 있는 주 파일입니다. |
| **.shx** | **Shapefile Index**: `.shp` 파일의 형상 데이터를 빠르게 검색하기 위한 인덱스 파일입니다. |
| **.dbf** | **dBase Database**: 지리적 형상에 대응하는 속성 정보(예: 행정동 명칭, 코드 등)를 표 형식으로 저장합니다. |
| **.prj** | **Projection**: 데이터의 좌표계(Coordinate System) 및 투영법 정보를 텍스트 형식으로 저장합니다. |
| **.cpg** | **Code Page**: `.dbf` 파일의 텍스트 인코딩 정보(예: UTF-8, CP949)를 명시합니다. |
| **.parquet** | **Apache Parquet**: 컬럼 기반의 효율적인 데이터 저장 형식으로, 대용량 데이터를 압축하여 빠르게 읽고 쓸 수 있습니다. |
| **.nc** | **NetCDF (Network Common Data Form)**: 다차원 배열 데이터를 저장하기 위한 형식으로, 기상 및 해양학 분야에서 위경도 기반 격자 데이터를 저장할 때 주로 사용됩니다. |
| **.md** | **Markdown**: 텍스트 기반의 문서 파일로, 현재 보고 계시는 설명 파일입니다. |

## 주요 파일 설명

### 1. 행정동 경계 데이터 (`BND_ADM_DONG_PG.*`)
- **설명**: 대한민국의 행정동 단위 경계 정보를 담고 있는 폴리곤(Polygon) 데이터입니다.
- **좌표계**: `KGD2002_Central_Belt_2010` (Korea Geodetic Datum 2002)
- **구성 파일**: `.shp`, `.shx`, `.dbf`, `.prj`, `.cpg`

### 2. 격자-법정동 매핑 데이터 (`grid_to_lawid.parquet`)
- **설명**: 기상청의 격자 지점과 법정동 ID(Law ID) 간의 관계를 정의한 데이터입니다. 기상 데이터를 특정 행정 구역별로 집계하거나 매핑할 때 사용됩니다.

### 3. 지상 격자 위경도 데이터 (`sfc_grid_latlon.nc`)
- **설명**: 지상(Surface) 격자 시스템의 각 지점에 대한 위도(Latitude)와 경도(Longitude) 정보를 담고 있는 데이터입니다.
